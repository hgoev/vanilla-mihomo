# vanilla-mihomo
windows mihomo裸核运行配置与一键执行脚本
**适用与系统代理模型，tun模型可忽略该脚本**
## 使用说明
- 下载[mihomo裸核](https://github.com/MetaCubeX/mihomo/releases)
- 解压到任一文件夹，并将命名mihomo.exe
- 自定义配置[config.yaml](https://github.com/MetaCubeX/mihomo/blob/Meta/docs/config.yaml)文件,配置文件中端口设置为7890，若更改，则[toggle_mihomo.vbs](https://github.com/hgoev/vanilla-mihomo/blob/main/toggle_mihomo.vbs)中36行端口也随之更改
- 配置完毕config文件后，双击toggle_mihomo.vbs即可
